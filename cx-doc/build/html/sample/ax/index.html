Use Case : Attribute Exchange¶

Attribute Excahnge Senario¶

In this senario, An End User is using a RP service and is asked to be authenticated with OpenID. The service needs the End User’s privacy information stored at the OP where his subject’s account is registred. Because the End User wants his privacy to be proteced under a contract binding all these stake holder, he choses OpenID CX protocol to share his profile attributes between the OP and the RP.
